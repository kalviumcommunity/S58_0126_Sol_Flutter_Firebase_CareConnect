CareConnect is a mobile-first app built with Flutter + Firebase to help parents discover trusted babysitters/caregivers with clearer profiles, safer access, and real-time updates.

## Sprint #2

**Theme:** Flutter + Firebase Mobile Application  
**Duration:** 4 weeks  
**Goal:** Deliver an MVP with authentication, database integration, and core UI screens.

## Problem

Parents looking for babysitters or caregivers often struggle to find a trusted and transparent platform. Existing options lack reliable verification, clear ratings, and real-time updates, which raises safety concerns and makes decision-making difficult.

## Solution

CareConnect aims to make childcare discovery safer, simpler, and more transparent by focusing on:

- Secure user authentication
- Clear and structured caregiver information
- Real-time data synchronization
- A clean, intuitive mobile experience for parents

## Tech Stack

- **Frontend:** Flutter
- **Backend/Cloud:** Firebase (Auth, Firestore)
- **Version Control:** Git + GitHub
