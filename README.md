# CareConnect (S58) - Flutter + Firebase

CareConnect is a mobile-first app built during a 4-week sprint to provide a clean and fast user experience with real-time data sync using Firebase.

## Tech Stack

- Flutter (Dart)
- Firebase (Auth + Firestore) (planned/in progress)
- Android Emulator for testing (Android 16 / API 36)

## Project Setup (Local)

### 1) Prerequisites

- Flutter SDK installed
- Android Studio + Android SDK installed
- A running Android Emulator (or Android device)

Check:

```bash
flutter doctor
flutter devices
```
